## 1. 项目背景

当前机加工车间的物料管理、生产日报统计、成本核算及质量分析工作主要依赖 Excel 手工记录（包括单价表、采购明细、领用记录、日产量表、质量达成率报表等）。随着车间规模扩大、产品种类增多，手工方式存在以下痛点：

- **数据分散**：采购、领用、生产、质量数据分布在多个独立 Excel 文件中，难以关联分析。
- **成本核算滞后**：移动加权平均成本需人工计算，容易出错且无法实时掌握库存价值。
- **生产进度不透明**：每日产量、良品/不良品、达成率、合格率统计繁琐，无法自动汇总班组/产品/员工绩效。
- **报表生成耗时**：月度成本报表、质量达成率报表、损耗分析报表需大量手工整理。

本项目旨在开发一套 **物料出入库管理 + 生产成本核算 + 生产日报与质量统计分析** 的一体化系统，实现数据电子化、流程规范化、统计自动化，提升管理效率与决策支持能力。

---

## 2. 项目目标

1. **实现材料全流程管理**：采购入库 → 库存移动加权平均 → 领用出库 → 期末结存。
2. **实现生产日报数字化**：支持 A/B 班组录入每日产量、良品/不良品、节拍、损失时间等，自动计算理论产量、合格率、达成率。
3. **自动生成统计报表**：
   - 月度成本汇总表（采购总额、领用总额、库存金额）
   - 产品质量达成率报表（按产品/机器/班组）
   - 班组达成率与合格率报表
   - 员工个人绩效报表
   - 产品完成进度报表（已做 vs 计划）
4. **提供基础数据维护功能**：机器列表（16台）、班组人员（可动态增删）、材料规格、产品主数据及月度计划量。
5. **系统需具备易用性**：前端界面简洁，支持下拉选择、自动计算、日期筛选，后端 API 稳定高效。

## 3. 功能需求

### 3.1 基础数据管理

| 模块         | 功能点         | 说明                                                         |
| ------------ | -------------- | ------------------------------------------------------------ |
| 材料规格管理 | 增、删、改、查 | 字段：材料规格（唯一）、单位、标准单价                       |
| 机器管理     | 增、删、改、查 | 预置 16 台机器（走芯机1-12号、马扎克、兄弟1-3号），支持增删  |
| 班组人员管理 | 增、删、改、查 | 人员属性：姓名、所属班组（A班 / B班），支持动态增减          |
| 产品主数据   | 增、删、改、查 | 字段：产品编号、产品名称、材料规格、当月计划数量（按月设置） |

### 3.2 采购入库（材料入库板块）

- **输入字段**：
  - 到货时间（日期选择）
  - 加工机器（下拉选择，来自机器列表）
  - 产品名称（手动输入）
  - 产品编号（手动输入）
  - 材料规格（下拉选择，来自材料规格）
  - 入库数量（根数）
  - 入库总重量（kg）
  - 实际单价（元/kg，手动输入）
  - 备注（可选）
- **自动计算**：总价 = 入库总重量 × 实际单价
- **业务规则**：保存采购记录后，自动更新对应材料的库存数量、库存金额、移动加权平均单价（加权平均算法）。

### 3.3 领用出库（出库板块）

- **输入字段**：
  - 出库日期（日期选择）
  - 加工机器（下拉）
  - 产品编号（手动）
  - 材料规格（下拉）
  - 领用数量（根数）
  - 领用总重量（kg）
  - 领用人（下拉，来自班组人员）
  - 备注（可选）
- **自动计算**：
  - 领用成本 = 领用总重量 × 当前加权平均单价（实时从库存获取）
- **业务规则**：
  - 领用数量不能超过当前库存，否则提示库存不足。
  - 保存领用记录后，扣减库存数量、金额，重新计算加权平均单价。

### 3.4 库存与成本核算

- **实时库存查询**：按材料规格展示当前库存总重量、库存总金额、加权平均单价。
- **月度成本报表**：
  - 按材料规格统计：期初数量/金额、本月采购数量/金额、本月领用数量/成本、期末数量/金额。
  - 总体汇总：本月采购总金额、本月领用总成本、期末库存总金额。

### 3.5 生产日报（机加工产量报表）

- **录入字段**：
  - 生产日期（日期选择）
  - 机器（下拉）
  - 产品名称
  - 产品编号
  - 材料规格
  - 生产节拍（秒/件）
  - 生产时间（秒）
  - 良品数量
  - 实绩数量（总产出）
  - 损失时间（分钟）
  - 班组（A班 / B班）
  - 操作工（根据班组动态下拉）
  - 计划数量（可选，也可从产品主数据读取）
- **自动计算**：
  - 理论产量 = 生产时间 ÷ 生产节拍
  - 不良数量 = 实绩数量 - 良品数量
  - 合格率 = 良品数量 ÷ 实绩数量（若实绩为0则合格率为0）
- **完成进度**：按产品累计当月良品数，对比计划数量，计算完成率。

### 3.6 统计报表

#### 3.6.1 产品质量达成率报表

- **维度**：产品 + 机器 + 班组
- **指标**：
  - 理论产量（A班累计 / B班累计 / 合计）
  - 实绩数量（A班累计 / B班累计 / 合计）
  - 良品数量（A班累计 / B班累计 / 合计）
  - 不良数量
  - 班组达成率 = 班组实绩 ÷ 班组理论
  - 班组合格率 = 班组良品 ÷ 班组实绩
  - 产品总达成率 = (A班实绩+B班实绩) ÷ (A班理论+B班理论)
  - 产品总合格率 = (A班良品+B班良品) ÷ (A班实绩+B班实绩)

#### 3.6.2 班组达成率报表

- 按 A班、B班 汇总所有产品的理论总量、实绩总量、良品总量，计算班组整体达成率、整体合格率。

#### 3.6.3 员工统计报表

- 按操作工姓名汇总：
  - 该员工参与生产的各产品理论产量、实绩、良品
  - 个人达成率 = 个人总实绩 ÷ 个人总理论
  - 个人合格率 = 个人总良品 ÷ 个人总实绩

#### 3.6.4 产品完成进度报表

- 按产品展示：当月计划数量、累计良品数量、完成率（累计良品/计划）
- 支持按月筛选。

#### 3.6.5 损耗分析（可选扩展）

- 按材料规格统计：领用总重量、成品产出总重量（需建立产品净重BOM）、损耗量、损耗率。

### 3.7 报表导出

所有报表均支持导出为 Excel 文件（.xlsx）。

## 4. 非功能需求

| 类别           | 要求                                                         |
| -------------- | ------------------------------------------------------------ |
| **性能**       | API 响应时间 ≤ 2 秒（95% 请求）；报表查询（含聚合）≤ 5 秒。  |

## 9. 核心业务规则与算法

### 9.1 移动加权平均法

采购入库时：
```
新库存总重量 = 原库存总重量 + 本次采购重量
新库存总金额 = 原库存总金额 + 本次采购金额
新加权平均单价 = 新库存总金额 / 新库存总重量
```
领用出库时：
```
领用成本 = 领用重量 × 当前加权平均单价
新库存总重量 = 原库存总重量 - 领用重量
新库存总金额 = 原库存总金额 - 领用成本
新加权平均单价 = 新库存总金额 / 新库存总重量（若库存为0则单价置0）
```

### 9.2 生产日报计算公式

- **理论产量** = 生产时间（秒） ÷ 生产节拍（秒/件）  
- **不良数量** = 实绩数量 - 良品数量  
- **合格率** = 良品数量 ÷ 实绩数量（实绩为0时合格率为0）  
- **产品完成率**（进度报表）= 当月累计良品数量 ÷ 当月计划数量  

### 9.3 班组/产品达成率

```
班组达成率 = 班组总实绩 ÷ 班组总理论产量
产品总达成率 = (A班实绩+B班实绩) ÷ (A班理论+B班理论)
合格率 = 良品数 ÷ 实绩数
```

### 9.4 库存扣减防超卖

在领用出库 API 中，使用 MongoDB 事务或带条件更新（如 `db.inventory.updateOne({material_spec: xxx, total_qty_kg: {$gte: issue_qty}}, ...)`），若更新失败则返回库存不足错误。

## 12. 验收标准

1. **采购入库**：能正确录入数据并自动更新库存加权平均单价，库存快照准确。
2. **领用出库**：能正确扣减库存，领用成本按当前均价计算，库存不足时拒绝出库。
3. **生产日报**：自动计算理论产量、不良数、合格率；支持按日期/机器/班组查询。
4. **成本报表**：月度采购总额、领用总额、期末库存金额与手工统计误差不超过 0.01%。
5. **质量达成率报表**：按产品/班组统计的达成率、合格率与 Excel 手工计算结果一致。
6. **班组/员工报表**：能正确汇总每人/每班的理论、实绩、良品数据。
7. **导出功能**：所有报表可导出为 Excel 文件，格式清晰。
8. **基础数据管理**：机器、材料、人员、产品计划可增删改查，下拉选项实时更新。
9. **权限控制**：不同角色登录后只能访问授权页面和操作。
